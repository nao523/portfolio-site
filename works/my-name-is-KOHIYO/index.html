My Name is KOHIYO

作品説明

  • URL

    https://nao523.github.io/my-name-is-KOHIYO/

  • サイト種類

    ブランディングサイト（キャラクター紹介）

  • 概要

    架空のキャラクター「コヒヨ」ファミリーの紹介サイトを制作しました。

  • 目的

    既存デザインをもとに模写コーディングを行い、HTML・CSSのレイアウト実装やレスポンシブ対応の理解を深めることを目的としました。

  • ターゲット

    かわいいキャラクター・動物コンテンツが好きな方

  • コーディング

    header / main / section などのセマンティックなHTMLを使用し、見た目だけでなく構造としてもわかりやすいマークアップを行っています。
    装飾画像と情報を持つ画像を適切に分け、alt 属性や見出し階層にも配慮し、アクセシビリティを意識した実装を行いました。
    レスポンシブ対応では、スマートフォンでも世界観を損なわずに読みやすくなるよう、余白・文字サイズ・装飾の見せ方を調整しています。

  • 制作期間

    コーディング 4時間

  • 使用ツール等

    Figma / html / css / Visual Studio Code
