名もなき中華料理屋

作品説明

  • URL

    https://nao523.github.io/nameless-chinese-restaurant/

  • サイト種類

    コーポレートサイト（中華料理店／1ページ構成・来店促進を目的）

  • 概要

    架空の中華料理店のサイトを制作しました。

  • 目的

    既存デザインをもとに模写コーディングを行い、HTML・CSSのレイアウト実装やレスポンシブ対応の理解を深めることを目的としました。

  • ターゲット

    近隣でランチや夕食に利用できる中華料理店を探している20〜50代の男女

  • コーディング

    セマンティックなHTMLを意識し、header / main / section など適切な要素を使用して構造を明確にしました。
    また、見出し階層の整理や alt 属性の設定、button と aタグの適切な使い分けを行い、アクセシビリティにも配慮しています。
    レスポンシブ対応では、スマートフォンでも読みやすく操作しやすいように文字サイズ・余白・行間を調整しました。

  • 制作期間

    コーディング 8時間

  • 使用ツール等

    Figma / html / css / Visual Studio Code
