シンプルブログサイト

作品説明

  • サイト種類

    メディアサイト（ライフスタイルブログ）

  • 概要

    シンプルなブログサイトを制作しました。WordPressでオリジナルテーマを作成してコーディングしています。
    またレスポンシブ対応・jsでハンバーガーメニューのアニメーション実装も行なっています。

  • 目的

    自分の学んだ知識をアウトプットすることを目的としました。

  • ターゲット

    Webデザイン等を勉強し始めた方

  • デザイン

    投稿内容をメインとするため、全体的にシンプルなデザインとしました。
    その中で背景色や文字色等は統一感を持たせるために設定しています。

  • コーディング

    セマンティックなHTMLを意識し、単に見た目を整えるだけでなく、検索エンジンやスクリーンリーダーが正しく情報を解釈できるよう、適切なHTMLタグ（header / main / section など）を選択しました。
    また、テキストと背景色のコントラスト比を 4.5:1 以上（AA準拠） になるようカラーパレットを調整し、可読性を最大化させています。
    レスポンシブ対応では、スマートフォンでも読みやすく操作しやすいように文字サイズ・余白・行間を調整しました。

  • 制作期間

    企画〜デザイン 7時間
    コーディング 9時間

  • 使用ツール等

    Figma / Wordpress / html / css / javascript / Visual Studio Code
